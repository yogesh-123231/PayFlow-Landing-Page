## 🛠 Installation & Usage

# Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/PayFlow-Landing-Page.git
cd PayFlow-Landing-Page
```
# Install dependencies
```
npm install
```
# Start development server
```
npm run dev
```
# Build for production
```
npm run build
```
